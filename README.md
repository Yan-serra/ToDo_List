Sistema de lista de tarefas (ToDo List)

•	Objetivo do programa:

Este programa tem como objetivo ajudar a organizar tarefas diárias do dia a dia em casa como: lavar louça, lavar roupa...  já que durante o dia pode aparecer imprevistos que requer atenção imediata e com isso acabar esquecendo ou deixando algum a fazer para trás. Com o sistema de lista de tarefas isso não seria problema já que ela teria tudo salvo e assim não deixar nada para trás.

•	Tecnologias utilizadas:

1.	HTML5
2.	CSS3
3.	JavaScript

•	Funções do sistema:

1.	Cadastro de tarefas
2.	Listagem de tarefas
3.	Pesquisa por título
4.	Marca a tarefa como concluída 
5.	Altera o status da tarefa entre concluída e pendente
6.	Edita tarefa
7.	Remove tarefa
8.	Armazenamento direto no navegador

•	Explicação do código:

No código está presente funções bem importantes como function addTarefa( ) para coletar os dados inseridos nos campos, função lista( ) para exibir as tarefas cadastradas na tela,  function toggleStatus(index) para alterar o status entre concluída e pendente,  function editarTarefa(index) para editar as tarefas posteriormente como o título e descrição da tarefa e função a function pesquisaTarefa para pode pesquisar a tarefa desejada pelo título

•	Modo de uso:

1.	Preencha os campos de título, descrição da tarefa e selecione o nível de prioridade da tarefa.
2.	Clique em “Adicionar a tarefa”.
3.	Use os botões editar, remover, concluir e alterar tarefas.
4.	Use o campo de “Pesquisa tarefa” pra pesquisar alguma tarefa especifica.
5.	Apague oque pesquisou no campo “Pesquisa tarefa” para voltar a listar todas as tarefas.
